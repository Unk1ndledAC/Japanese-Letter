# 日本書簡格式指南
# Japanese Letter Format Guide

## 1. 書簡の基本構造

日語書簡大致可分為以下五個部分：

| 部分 | 日語 | 説明 |
|------|------|------|
| 前文 | 前文（ぜんぶん）| 頭語 + 時候の挨拶 + 安否の挨拶 |
| 主文 | 主文（しゅぶん）| 書簡的核心內容 |
| 末文 | 末文（まつぶん）| 結びの挨拶 |
| 結語 | 結語（けつご）| 敬具・草々等 |
| 後付 | 後付（こうづけ）| 日付・署名・宛名 |

### 1.1 前文（Zenbun）

前文包含以下內容：

#### 頭語（Topigō）

書信的開頭語，常用的有：

| 頭語 | 用途 | 對應結語 |
|------|------|----------|
| 拝啓 | 一般書信 | 敬具 |
| 拝復 | 回信 | 敬具 |
| 啓上 | 鄭重 | 敬具 |
| 再啓 | 回信（較正式）| 敬具 |
| 前略 | 省略前文 | 草々 |
| 冠省 | 省略前文 | 草々 |
| 略啓 | 省略前文 | 草々 |

#### 時候の挨拶（Sekino Aisatsu）

根據季節和月份的問候語：

**1月〜3月（春 spring）**
| 月 | 時候の挨拶 |
|----|------------|
| 1月 | 新春の候・厳寒の候・酷寒の候 |
| 2月 | 立春の候・晩冬の候・余寒の候 |
| 3月 | 早春の候・浅春の候・春暖の候 |

**4月〜6月（夏 early summer）**
| 月 | 時候の挨拶 |
|----|------------|
| 4月 | 春暖の候・春kówの候・花好みの候 |
| 5月 | 新緑の候・若葉の候・薫風の候 |
| 6月 | 梅雨の候・初夏の候・ 向夏の候 |

**7月〜9月（夏 late summer）**
| 月 | 時候の挨拶 |
|----|------------|
| 7月 | 盛夏の候・酷暑の候・灼熱の候 |
| 8月 | 残暑の候・晩夏の候・葉曇の候 |
| 9月 | 初秋の候・新秋の候・秋凉の候 |

**10月〜12月（秋・冬）**
| 月 | 時候の挨拶 |
|----|------------|
| 10月 | 秋涼の候・錦秋の候・紅葉の候 |
| 11月 | 晩秋の候・深秋の候・紅葉の候 |
| 12月 | 初冬の候・寒冷の候・師走の候 |

#### 安否の挨拶（Anpino Aisatsu）

詢問對方近況的問候語：
- 各位にはますますご清栄のこととお慶び申し上げます
- 各位にはますますご健勝のこととお慶び申し上げます
- ご夫妻にはますますご清栄のこととお慶び申し上げます

### 1.2 主文（Shubun）

書信的正文部分，用於表達主要內容。

常用引入詞：
- さて
- ところで
- つきましては
- 、早速ですが
- 実は

### 1.3 末文（Matsubun）

結束正文時的問候語：
- 末筆ながら、○○をお祈り申し上げます
- 取り乱しいたしますが、○○

### 1.4 結語（Ketsugo）

與頭語對應的結束語：

| 頭語 | 結語 |
|------|------|
| 拝啓 | 敬具 |
| 拝復 | 敬具 |
| 啓上 | 敬具 |
| 再啓 | 敬具 |
| 前略 | 草々 |
| 冠省 | 草々 |

### 1.5 後付（Kōzuke）

#### 日付

和曆或西曆：
- 令和6年12月1日
- 2024年12月1日

#### 署名

一般書信用「（署名）」，正式書信用「（印）」。

#### 宛名

在書信最後一行寫上收件人的姓名：
- ○○○ 様（對一般人）
- ○○○ 先生（對老師）
- ○○○ 殿（非常正式）

## 2. 封筒格式

### 2.1 直式封筒（縦書き封筒）

#### 表面
```
┌─────────────────────────────┐
│                   123-4567   │ ← 受信人邮编
│                   ████████  │ ← 受信人地址
│                   ████████  │
│                   ○○○ 様    │ ← 受信人氏名
│                              │
│                              │
└─────────────────────────────┘
```

#### 裏面
```
┌─────────────────────────────┐
│  令和6年12月1日              │ ← 日期
│                              │
│                              │
│  100-8111        〆          │ ← 差出人邮编・封字
│  ████████                    │
│  ████████                    │
│  ○○○                         │ ← 差出人情報
└─────────────────────────────┘
```

### 2.2 横式封筒（洋封筒）

#### 表面
```
┌─────────────────────────────┐
│ ┌──────────┐                │
│ │差出人情報 │      切手       │ ← 左上：差出人 / 右上：切手
│ │邮编      │                │
│ │地址      │                │
│ └──────────┘                │
│                    123-4567 │ ← 受信人邮编
│                    ████████ │ ← 受信人地址
│                    ████████ │
│                    ○○○ 様   │ ← 受信人氏名
│                              │
└─────────────────────────────┘
```

#### 裏面
```
┌─────────────────────────────┐
│                              │
│                              │
│           〆                 │ ← 封字
│                              │
│         ○○○ 様               │
│       ████████               │
│       ████████               │
│       100-8111               │
└─────────────────────────────┘
```

## 3. 常用書簡文例

### 3.1 年始の挨拶

```
拝啓
新春の候、愈々ご清栄のこととお慶び申し上げます。
旧年中は格別のお引き立てを賜り、厚く御礼申し上げます。
本年も倍旧の努力誠心誠意努める所存でございます。
何卒倍旧のご支援を賜りますようお願い申し上げます。
敬具

令和6年元旦
```

### 3.2 見舞いの手紙

```
啓上
酷暑の候、愈々ご清栄のこととお慶び申し上げます。
○○様にはいかがお過ごしでしょうか。
お蔭様で私ども全家共に元気に暮らしております。
稍将自己養生ください。
敬具
```

### 3.3 お悔み状

```
啓上
○○様の逝去の報に接し、的心からご冥福をお祈り申し上げます。
生前中の御功績に敬意を表します。
ここに謹んで哀悼の意を表します。
敬具
```
